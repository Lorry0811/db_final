## 1. 認證相關功能

### 1.1 註冊帳號

**操作步驟：**
1. 點擊導航欄右上角的「註冊」按鈕
2. 進入註冊頁面 (`/register`)
3. 填寫以下資訊：
   - **電子郵件**：輸入有效的電子郵件地址
   - **使用者名稱**：輸入想要使用的使用者名稱
   - **密碼**：輸入密碼
   - **確認密碼**：再次輸入密碼以確認
4. 點擊「註冊」按鈕
5. 註冊成功後會自動跳轉到登入頁面

**注意事項：**
- 密碼和確認密碼必須一致
- 所有欄位都是必填的

### 1.2 登入系統

**操作步驟：**
1. 點擊導航欄右上角的「登入」按鈕
2. 進入登入頁面 (`/login`)
3. 輸入以下資訊：
   - **電子郵件**：輸入註冊時使用的電子郵件
   - **密碼**：輸入密碼
4. 點擊「登入」按鈕
5. 登入成功後：
   - 一般使用者會跳轉到「我的儀表板」頁面 (`/dashboard`)
   - 管理員會跳轉到「管理後台」頁面 (`/admin`)

**注意事項：**
- 登入後，導航欄右上角會顯示使用者名稱和「登出」按鈕
- 管理員的使用者名稱旁會顯示「管理員」標籤

### 1.3 登出系統

**操作步驟：**
1. 確認已登入（導航欄右上角顯示使用者名稱）
2. 點擊導航欄右上角的「登出」按鈕
3. 系統會清除登入狀態並跳轉到首頁
4. 登出後，導航欄右上角會恢復顯示「登入」和「註冊」按鈕

## 2. 刊登管理功能

### 2.1 新增刊登

**操作步驟：**
1. 點擊導航欄的「刊登商品」連結，或從首頁點擊「刊登商品」按鈕
2. 進入新增刊登頁面 (`/postings/new`)
3. 填寫刊登資訊：
   - **標題**：輸入商品標題（必填）
   - **描述**：輸入商品詳細描述（選填）
   - **價格**：輸入商品價格，單位為新台幣（必填，必須 ≥ 0）
   - **分類**：從下拉選單選擇商品分類（選填）
   - **課程**：在搜尋框中輸入關鍵字搜尋相關課程，然後從下拉選單選擇（選填）
   - **圖片**：點擊「選擇圖片」按鈕，從本機選擇 1-5 張圖片上傳
4. 點擊「建立刊登」按鈕
5. 刊登成功後會自動跳轉到該刊登的詳情頁面

**注意事項：**
- 必須先登入才能新增刊登
- 圖片支援 JPG、PNG、GIF 格式
- 圖片會自動壓縮以節省空間
- 可以上傳多張圖片，系統會按順序顯示

### 2.2 編輯自己的刊登

**操作步驟：**
1. 進入要編輯的刊登詳情頁面 (`/postings/[id]`)
2. 確認該刊登是您自己建立的（會顯示「編輯」按鈕）
3. 點擊「編輯」按鈕
4. 進入編輯頁面 (`/postings/[id]/edit`)
5. 修改需要更新的欄位（標題、描述、價格、分類、課程、圖片）
6. 點擊「更新刊登」按鈕
7. 更新成功後會自動跳轉到該刊登的詳情頁面

**注意事項：**
- 只能編輯自己建立的刊登
- 圖片可以重新上傳，舊圖片會被替換

### 2.3 刪除/下架自己的刊登

**操作步驟：**
1. 進入「我的儀表板」頁面 (`/dashboard`)
2. 在「我的刊登」區塊中找到要下架的刊登
3. 點擊該刊登的「下架」按鈕
4. 確認下架操作
5. 下架後，該刊登狀態會變為「已下架」，不再顯示在商品列表頁面

**注意事項：**
- 只能下架自己建立的刊登
- 已售出的刊登無法下架

### 2.4 查看自己的所有刊登

**操作步驟：**
1. 點擊導航欄的「我的儀表板」連結
2. 進入儀表板頁面 (`/dashboard`)
3. 在「我的刊登」區塊中查看所有刊登
4. 可以使用狀態篩選器篩選：
   - **全部**：顯示所有狀態的刊登
   - **刊登中**：只顯示正在刊登的商品
   - **已售出**：只顯示已售出的商品
   - **已下架**：只顯示已下架的商品

**顯示資訊：**
- 刊登標題
- 價格
- 狀態
- 建立時間
- 操作按鈕（編輯、下架）

## 3. 瀏覽與搜尋功能

### 3.1 瀏覽所有刊登

**操作步驟：**
1. 點擊導航欄的「瀏覽商品」連結，或從首頁點擊「瀏覽商品」按鈕
2. 進入商品列表頁面 (`/postings`)
3. 系統會顯示所有「刊登中」狀態的商品
4. 商品以卡片形式展示，包含：
   - 商品圖片（第一張）
   - 商品標題
   - 價格
   - 分類名稱
   - 課程名稱（如果有）
   - 建立時間
5. 點擊商品卡片可進入該商品的詳情頁面

### 3.2 搜尋刊登

**操作步驟：**
1. 進入商品列表頁面 (`/postings`)
2. 在搜尋表單的「搜尋關鍵字」欄位輸入關鍵字
3. 點擊「搜尋」按鈕
4. 系統會顯示標題或描述中包含關鍵字的商品

### 3.3 依分類篩選

**操作步驟：**
1. 進入商品列表頁面 (`/postings`)
2. 在搜尋表單的「分類」下拉選單中選擇想要的分類
3. 點擊「搜尋」按鈕
4. 系統會只顯示該分類的商品

### 3.4 依課程篩選

**操作步驟：**
1. 進入商品列表頁面 (`/postings`)
2. 在搜尋表單的「課程」下拉選單中選擇想要的課程
3. 點擊「搜尋」按鈕
4. 系統會只顯示該課程相關的商品

### 3.5 依價格範圍篩選

**操作步驟：**
1. 進入商品列表頁面 (`/postings`)
2. 在搜尋表單的「最低價格」欄位輸入最低價格
3. 在「最高價格」欄位輸入最高價格
4. 點擊「搜尋」按鈕
5. 系統會只顯示價格在指定範圍內的商品

### 3.6 分頁功能

**操作步驟：**
1. 在商品列表頁面底部會顯示分頁按鈕
2. 點擊頁碼數字可跳轉到對應頁面
3. 每頁預設顯示 20 筆商品

### 3.7 查看刊登詳情

**操作步驟：**
1. 在商品列表頁面或首頁點擊商品卡片
2. 進入商品詳情頁面 (`/postings/[id]`)
3. 頁面會顯示：
   - 商品圖片（可切換查看多張）
   - 商品標題
   - 價格
   - 狀態（刊登中/已售出等）
   - 描述
   - 分類和課程資訊
   - 賣家資訊（使用者名稱，可點擊查看賣家評價）
   - 建立時間和更新時間
4. 在詳情頁面可以進行以下操作：
   - 購買商品（如果狀態為「刊登中」）
   - 收藏/取消收藏
   - 發送私訊給賣家
   - 舉報不當刊登
   - 查看和新增留言
   - 編輯/刪除自己的刊登（如果是自己的刊登）

## 4. 互動功能

### 4.1 收藏刊登

**操作步驟：**
1. 進入商品詳情頁面 (`/postings/[id]`)
2. 點擊「收藏」按鈕（心形圖示）
3. 按鈕會變為「已收藏」狀態（紅色心形）
4. 該商品會被加入您的收藏列表

**取消收藏：**
1. 在商品詳情頁面點擊「已收藏」按鈕
2. 按鈕會變回「收藏」狀態
3. 該商品會從您的收藏列表中移除

### 4.2 查看我的收藏列表

**操作步驟：**
1. 點擊導航欄的「我的儀表板」連結
2. 在儀表板頁面點擊「我的收藏」按鈕
3. 或直接點擊導航欄的「我的收藏」連結（如果有的話）
4. 進入收藏列表頁面 (`/favorites`)
5. 頁面會顯示所有您收藏的商品
6. 點擊商品卡片可進入商品詳情頁面
7. 在收藏列表頁面也可以取消收藏

### 4.3 在刊登下方留言

**操作步驟：**
1. 進入商品詳情頁面 (`/postings/[id]`)
2. 滾動到頁面下方的「留言區」
3. 在留言輸入框中輸入留言內容
4. 點擊「送出留言」按鈕
5. 留言會立即顯示在留言列表中

**注意事項：**
- 必須先登入才能留言
- 留言內容不能為空

### 4.4 查看刊登的所有留言

**操作步驟：**
1. 進入商品詳情頁面 (`/postings/[id]`)
2. 滾動到頁面下方的「留言區」
3. 系統會自動顯示該商品的所有留言
4. 每則留言會顯示：
   - 留言者使用者名稱
   - 留言內容
   - 留言時間
   - 編輯/刪除按鈕（如果是自己的留言）

### 4.5 編輯自己的留言

**操作步驟：**
1. 進入商品詳情頁面 (`/postings/[id]`)
2. 在留言區找到自己的留言
3. 點擊該留言的「編輯」按鈕
4. 修改留言內容
5. 點擊「儲存」按鈕
6. 留言會更新為新內容

**注意事項：**
- 只能編輯自己的留言

### 4.6 刪除自己的留言

**操作步驟：**
1. 進入商品詳情頁面 (`/postings/[id]`)
2. 在留言區找到自己的留言
3. 點擊該留言的「刪除」按鈕
4. 確認刪除操作
5. 留言會被移除

**注意事項：**
- 只能刪除自己的留言

## 5. 交易功能

### 5.1 購買商品

**操作步驟：**
1. 進入商品詳情頁面 (`/postings/[id]`)
2. 確認商品狀態為「刊登中」
3. 確認您的餘額足夠（可在「我的儀表板」查看餘額）
4. 點擊「購買」按鈕
5. 系統會顯示確認對話框，顯示：
   - 商品標題
   - 價格
   - 您的餘額
6. 確認購買資訊無誤後，點擊「確認購買」
7. 購買成功後：
   - 系統會自動從您的餘額扣除商品價格
   - 商品價格會轉入賣家帳戶
   - 商品狀態會變為「已售出」
   - 會自動建立訂單
   - 購買按鈕會變為「已售出」並被禁用

**注意事項：**
- 必須先登入才能購買
- 不能購買自己的商品
- 餘額不足時無法購買
- 已售出的商品無法購買

### 5.2 查看我的訂單列表

**操作步驟：**
1. 點擊導航欄的「我的儀表板」連結
2. 在儀表板頁面點擊「我的訂單」按鈕
3. 或直接點擊導航欄的「我的訂單」連結（如果有的話）
4. 進入訂單列表頁面 (`/orders`)
5. 頁面會顯示所有您作為買家的訂單
6. 每筆訂單會顯示：
   - 訂單編號
   - 商品標題
   - 成交價格
   - 訂單狀態（已完成/已取消）
   - 訂單日期
7. 點擊訂單可進入訂單詳情頁面

### 5.3 查看訂單詳情

**操作步驟：**
1. 進入訂單列表頁面 (`/orders`)
2. 點擊想要查看的訂單
3. 進入訂單詳情頁面 (`/orders/[id]`)
4. 頁面會顯示：
   - 訂單編號
   - 訂單狀態
   - 訂單日期
   - 商品資訊（標題、描述、圖片）
   - 成交價格
   - 賣家資訊（使用者名稱、電子郵件）
   - 買家資訊（您自己）
5. 在訂單詳情頁面可以進行以下操作：
   - 對賣家進行評價（如果尚未評價）
   - 編輯/刪除已寫的評價
   - 舉報逃單（如果賣家未寄送商品）

### 5.4 儲值（增加餘額）

**操作步驟：**
1. 點擊導航欄的「我的儀表板」連結
2. 在儀表板頁面點擊「儲值」按鈕
3. 或直接點擊導航欄的「儲值」連結（如果有的話）
4. 進入儲值頁面 (`/topup`)
5. 在「儲值金額」欄位輸入想要儲值的金額
6. 點擊「確認儲值」按鈕
7. 儲值成功後，您的餘額會增加對應金額
8. 系統會自動建立一筆「儲值」類型的交易紀錄

**注意事項：**
- 儲值金額必須大於 0
- 儲值後餘額會立即更新

### 5.5 查看交易紀錄

**操作步驟：**
1. 點擊導航欄的「我的儀表板」連結
2. 在儀表板頁面點擊「交易紀錄」按鈕
3. 或直接點擊導航欄的「交易紀錄」連結（如果有的話）
4. 進入交易紀錄頁面 (`/transactions`)
5. 頁面會顯示所有交易紀錄，包含：
   - 交易類型（購買、售出、儲值）
   - 交易金額
   - 交易時間
   - 相關訂單編號（如果是購買/售出）
6. 可以使用「交易類型」下拉選單篩選：
   - **全部**：顯示所有類型的交易
   - **購買**：只顯示購買商品的交易
   - **售出**：只顯示售出商品的交易
   - **儲值**：只顯示儲值的交易

### 5.6 查看餘額

**操作步驟：**
1. 點擊導航欄的「我的儀表板」連結
2. 在儀表板頁面的「帳戶資訊」區塊中會顯示當前餘額
3. 餘額會以「NT$ X,XXX」的格式顯示

**其他查看方式：**
- 在儲值頁面 (`/topup`) 也會顯示當前餘額
- 在個人資料頁面 (`/profile`) 的統計資訊中會顯示餘額

## 6. 評價功能

### 6.1 對賣家進行評價（購買後）

**操作步驟：**
1. 完成一筆購買後，進入該訂單的詳情頁面 (`/orders/[id]`)
2. 在「評價賣家」區塊中，如果尚未評價，會顯示評價表單
3. 填寫評價資訊：
   - **評分**：選擇 1-5 顆星（必填）
   - **評價內容**：輸入評價文字（選填）
4. 點擊「提交評價」按鈕
5. 評價成功後會顯示在頁面上

**注意事項：**
- 只有購買後才能評價賣家
- 每個訂單只能評價一次
- 可以編輯或刪除已寫的評價

### 6.2 查看賣家的評價

**操作步驟：**
1. 進入商品詳情頁面 (`/postings/[id]`)
2. 在賣家資訊區塊中，點擊賣家名稱旁的「查看評價」連結
3. 進入賣家評價頁面 (`/users/[id]/reviews`)
4. 頁面會顯示：
   - 賣家的平均評分（1-5 顆星）
   - 所有評價列表，包含：
     - 評分
     - 評價內容
     - 評價者使用者名稱
     - 評價時間

**其他查看方式：**
- 在訂單詳情頁面也可以看到賣家的評價資訊

### 6.3 查看自己的評價紀錄

**操作步驟：**
1. 點擊導航欄的「我的評價」連結（如果有的話）
2. 或直接進入評價頁面 (`/reviews`)
3. 頁面會顯示所有您寫過的評價
4. 每則評價會顯示：
   - 被評價的使用者名稱
   - 評分
   - 評價內容
   - 評價時間
   - 相關訂單編號
5. 可以點擊「編輯」或「刪除」按鈕修改或移除評價

## 7. 私訊功能

### 7.1 發送私訊給賣家/買家

**方式一：從商品詳情頁面發送**
1. 進入商品詳情頁面 (`/postings/[id]`)
2. 點擊「發送私訊」按鈕
3. 會彈出私訊對話框
4. 在「訊息內容」欄位輸入想要發送的訊息
5. 點擊「發送」按鈕
6. 訊息會發送給該商品的賣家

**方式二：從私訊列表頁面發送**
1. 進入私訊列表頁面 (`/messages`)

3. 進入對話頁面 (`/messages/[userId]`)
4. 在頁面底部的輸入框中輸入訊息
5. 點擊「發送」按鈕或按 Enter 鍵
6. 訊息會立即顯示在對話中

**方式三：從訂單詳情頁面發送**
1. 進入訂單詳情頁面 (`/orders/[id]`)
2. 點擊「發送私訊給賣家」按鈕（如果是買家）
3. 或點擊「發送私訊給買家」按鈕（如果是賣家）
4. 會跳轉到對話頁面並自動填入收件人

**注意事項：**
- 必須先登入才能發送私訊
- 不能發送私訊給自己

### 7.2 查看私訊列表

1. 點擊導航欄的「私訊」連結

   - 訊息時間
   - 未讀訊息數量（如果有）

### 7.3 查看私訊詳情

1. 進入私訊列表頁面 (`/messages`)
2. 點擊想要查看的對話
3. 進入對話詳情頁面 (`/messages/[userId]`)

### 7.4 標記已讀/未讀

6. 點擊「提交」按鈕

- 不能舉報自己的留言

   - **舉報賣家**：如果賣家未寄送商品
   - **舉報買家**：如果買家未付款（此情況較少見，因為系統會自動扣款）

4. 每筆舉報會顯示：
   - 舉報類型（刊登舉報/留言舉報/逃單舉報）

   - 舉報狀態（待審核/已通過/已駁回）
   - 舉報時間
5. 可以使用篩選器篩選：
   - **狀態篩選**：全部/待審核/已通過/已駁回
   - **類型篩選**：全部/刊登舉報/留言舉報/逃單舉報

4. 頁面會顯示所有舉報的列表
5. 每個舉報會顯示：
   - 舉報類型（刊登舉報/留言舉報/逃單舉報）
   - 舉報者使用者名稱
   - 舉報對象（商品標題/留言內容/訂單編號）
   - 舉報狀態（待審核/已通過/已駁回）
   - 舉報時間
   - 操作按鈕（查看詳情）

#### 10.6.2 篩選舉報

**操作步驟：**
1. 進入舉報審核頁面 (`/admin/reports`)
2. 可以使用以下篩選器：
   - **狀態篩選**：全部/待審核/已通過/已駁回
   - **類型篩選**：全部/刊登舉報/留言舉報/逃單舉報
3. 選擇篩選條件後，列表會自動更新

#### 10.6.3 搜尋舉報

**操作步驟：**
1. 進入舉報審核頁面 (`/admin/reports`)
2. 在頁面頂部的搜尋框中輸入關鍵字
3. 點擊「搜尋」按鈕或按 Enter 鍵
4. 系統會顯示符合關鍵字的舉報（搜尋範圍包括舉報原因、商品標題、留言內容）

#### 10.6.4 查看舉報詳情

**操作步驟：**
1. 進入舉報審核頁面 (`/admin/reports`)
2. 找到想要查看的舉報
3. 點擊該舉報的「查看詳情」按鈕
4. 進入舉報詳情頁面 (`/admin/reports/[id]`)
5. 頁面會顯示：
   - **舉報資訊**：舉報類型、舉報者、舉報原因、舉報時間、舉報狀態
   - **被舉報內容**：
     - 如果是刊登舉報：顯示商品詳情
     - 如果是留言舉報：顯示留言內容和相關商品
     - 如果是逃單舉報：顯示訂單詳情和相關使用者
   - **審核操作**：通過、駁回、移除刊登按鈕

#### 10.6.5 審核舉報（通過/駁回）

**操作步驟：**
1. 進入舉報詳情頁面 (`/admin/reports/[id]`)
2. 仔細查看舉報內容和被舉報的內容
3. 決定審核結果：
   - **通過**：如果舉報內容屬實，點擊「通過」按鈕
   - **駁回**：如果舉報內容不屬實，點擊「駁回」按鈕
4. 系統會顯示確認對話框
5. 確認審核操作
6. 審核成功後：
   - 如果通過：被舉報的使用者違規次數會增加，系統可能會自動封鎖（如果違規次數達到上限）
   - 如果駁回：舉報狀態會更新為「已駁回」，不會影響被舉報的使用者

#### 10.6.6 處理違規（移除刊登、警告使用者）

**操作步驟：**
1. 進入舉報詳情頁面 (`/admin/reports/[id]`)
2. 如果舉報類型是「刊登舉報」，可以選擇移除該刊登：
   - 點擊「移除刊登」按鈕
   - 確認移除操作
   - 移除成功後，該刊登狀態會變為「已移除」
3. 審核通過後，系統會自動：
   - 增加被舉報使用者的違規次數
   - 如果違規次數達到上限（例如 3 次），系統會自動封鎖該使用者

**注意事項：**
- 移除刊登後，該刊登會從商品列表頁面消失
- 違規次數會影響使用者的帳戶狀態
- 系統會根據違規次數自動決定是否封鎖使用者

### 10.7 統計報表

#### 10.7.1 查看平台統計

**操作步驟：**

   - **交易統計**：總訂單數、總交易筆數、總營收

3. 頁面會顯示所有分類的統計資料
4. 每個分類會顯示：
   - 分類名稱
   - 該分類下的刊登總數
   - 該分類下的已售出數
   - 該分類下的總營收

#### 10.7.3 查看課程統計

4. 每個課程會顯示：
   - 課程代碼和課程名稱
   - 該課程相關的刊登總數
   - 該課程相關的已售出數
   - 該課程相關的總營收

#### 10.7.4 查看交易統計

3. 頁面會顯示交易相關的統計資料，包含：
   - **總體統計**：總訂單數、總交易金額、平均訂單金額
   - **交易類型統計**：購買交易數、售出交易數、儲值交易數
   - **時間統計**：今日交易、本週交易、本月交易（如果有的話）

## 📝 補充說明

### 導航欄功能

導航欄位於頁面頂部，包含以下連結：
- **首頁**：返回首頁
- **瀏覽商品**：進入商品列表頁面
- **刊登商品**：進入新增刊登頁面（需登入）
- **我的儀表板**：進入使用者儀表板（需登入）
- **私訊**：進入私訊列表頁面（需登入）
- **我的舉報**：進入舉報紀錄頁面（需登入）
- **個人資料**：進入個人資料頁面（需登入）
- **管理後台**：進入管理後台（僅管理員可見）
- **登入/註冊**：未登入時顯示
- **使用者名稱 + 登出**：登入後顯示

### 權限說明

- **一般使用者**：可以使用所有使用者功能，但無法進入管理後台
- **管理員**：可以使用所有功能，包括管理後台的所有功能
- **未登入使用者**：只能瀏覽商品和查看商品詳情，無法進行購買、留言、收藏等操作

### 常見問題

**Q: 為什麼我無法購買商品？**
A: 請確認：
1. 您已登入
2. 商品狀態為「刊登中」
3. 您的餘額足夠
4. 該商品不是您自己刊登的

**Q: 如何查看我的餘額？**
A: 進入「我的儀表板」頁面，在「帳戶資訊」區塊中會顯示當前餘額。

**Q: 如何成為管理員？**
A: 管理員帳號需要由系統管理員手動設定，一般使用者無法自行升級為管理員。

**Q: 為什麼我的刊登沒有顯示在商品列表頁面？**
A: 請確認：
1. 刊登狀態為「刊登中」
2. 刊登沒有被管理員移除
3. 您的帳戶沒有被封鎖

**Q: 如何聯繫賣家？**
A: 在商品詳情頁面點擊「發送私訊」按鈕，或進入訂單詳情頁面點擊「發送私訊給賣家」按鈕。
